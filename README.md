# frontProjetoFinal

Parte Front-End do Projeto Spring para o Projeto Cubo

## Aposta e Ganhe

Aplicativo que simula um jogo de apostas onde é escolhido um apostador vencedor do jogo, 
aleatoriamente. Feito utilizando o Angular.
